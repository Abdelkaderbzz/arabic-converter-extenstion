# Arabizzi — الإصدار 1.3.0

## ما الجديد

- **إخراج بالإنجليزية والفرنسية** — ترجمة Arabizi إلى الإنجليزية أو الفرنسية
- **محسّن لاختيار نوع الإخراج** — فصحى، عامية، إنجليزي، فرنسي
- **تبويبات السجل المحسّنة** — الأخيرة / المحفوظة
- **حقل إخراج قابل للتمرير** للنصوص الطويلة
- **تحديثات موقع arabizzi.com** — مراجعات، أسئلة شائعة، حركات تمرير
